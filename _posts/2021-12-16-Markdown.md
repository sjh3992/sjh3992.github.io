# What is Markdown
* 일반 텍스트 기반의 경량 마크업 언어
* 일반 텍스트로 서식이 있는 문서를 작성할 수 있다
* 일반 마크업 언어에 비해 문법이 쉽고 간단

# Basic Syntax of Markdown
* 제목(header) 작성
```markdown
# Header 1
## Header 2
### Header 3
#### Header 4
```

* 기울임체(*italic*) 작성
```markdown
*italic*
_italic_
```

* 강조체(**bold**) 작성
```markdown
**bold**
__bold__
```

* 취소선(~~strikethrough~~) 작성
```markdown
~~strikethrough~~
```

* 순서없는 리스트(unordered list) 작성
```markdown
- Hello World!
* Bye World!
```

* 순서 리스트(ordered list) 작성
```markdown
1. Hello World!
2. Bye World!
```

* 코드(Code) 작성
```markdown
`print()`
```
